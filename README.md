# AnyAuction

> A simple online auction marketplace

## Team members
- Sean Lussier
- Matthew Pietracupa
- Fahad Malik

---

## Description
**AnyAuction** is a user-friendly platform where people can **buy and sell products through auctions**.  
It connects buyers with sellers seamlessly.

---

## Who It's For
| Users   | Description |
|--------|------------|
| Buyers | Find great deals on items, or search for whatever you need. |
| Vendors | Selling items quickly or look to profit from reselling purchased items. |

---

## Key Features
- Auction & bidding system, with autobidding   
- Secure payment processing
- Lazy image loading/ automatic image configuration
- Snipe protection  
- Text bidding 
- Watchlist
